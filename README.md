## Dataset

An excel file extracted from CoinMarketCap in different ways: using cryptocmd python library and an extracted file saved on Github [here](https://github.com/soulayman-al-abdallah/public-repo.git)

Unstable coins, SOL FTT BITCOIN: 1210 rows × 21 columns

Stable coins, USDT & USDC:   1514 rows × 9 columns
